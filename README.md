# RNestarguars

##Getting Started  
###install dependencies. 
```
yarn install
```

###run on iOS simulator
```
npx pod-install
yarn ios
```
###run on android emulator
```
yarn android
```

##e2e testing
###install dependencies
```
brew tap wix/brew
brew install applesimutils
npm install -g detox-cli
```
